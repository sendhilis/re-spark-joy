# Lipafo — AI-powered wallet

Lipafo is an AI-powered wallet and switch platform built for the Kenyan market.

## Local development

```sh
npm install
npm run dev
```

## Tech stack

- React 18 + Vite + TypeScript
- Tailwind CSS
- shadcn/ui
- Managed backend (Auth, Database, Edge Functions, Realtime, Storage)

## Deployment

The application is published at <https://lipafo.wematech.africa>.

## Custom domain

Production traffic is served from `lipafo.wematech.africa`. DNS is managed by the Lipafo team.
